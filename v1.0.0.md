## step-by-step dev plan
1. lock release baseline by keeping typecheck/lint/test green on every change.
2. run high-risk flow e2e first: collaboration, step runner, plan/act, sessions.
3. run provider-backed smoke checks (openai + at least one secondary provider).
4. harden security/test automation and verify clear failure reporting in ui.
5. validate performance monitor metrics and threshold recommendations.
6. validate deployment wizard + rollback in provider sandboxes.
7. validate plugin marketplace install/uninstall + manifest safety.
8. close docs/release notes, then freeze for final regression pass.
